## Project4_Kashaf

1. Create a python virtual environment in codespaces
2. Create a python Scaffold (Create empty files: 'Makefile', 'requirements.txt', 'main.py', 'Dockerfile', 'mylib/__ init __.py')
3. Populate Makefile
4. Set up Continuous Integration 
5. Build a command line tool (cli-fire.py) using python fire library and use chmod +x cli-fire.py to make it executable, and then we can pass it as ./cli-fire.py --help to test the logic
